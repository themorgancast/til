# New Stuff in React 18 and 19

## As of React 18
- Automatic batching to group multiple states into a single re-render

## As of React 19
- Actions to streamline and automatically handle async function handling and state updates
- `useOptimistic` hook to show final state while async data mutations are happening
- `useActionState` hook
- `use()`
- Access `ref` as a prop
- Consolidated error reporting
